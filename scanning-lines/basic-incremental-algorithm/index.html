Basic Incremental Algorithm

This sample code implements the basic line drawing algorithm, explained in the book Computer Graphics: Principles and Practice under Section 3.2.1.

Pros

  • Basic and straightforward implemention of line drawing

Cons

  • Lots of floating-point arithmetic including rounding and, since the slope is a fraction it is inevitible.
  • Limited to cases where the slope is between -1 and 1, and the starting point should be the leftmost point. (Some modification required to support such cases)
  • Not possible implement a direction. Same set of points will be generated when the endpoints are reversed.

Usage

  • Move your mouse to change end point, left click to set start point of the line.
  • Select different algorithms and grid size to experiment.
