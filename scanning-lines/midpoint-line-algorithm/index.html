Midpoint Line Algorithm

This sample code implements the midpoint line drawing algorithm, explained in the book Computer Graphics: Principles and Practice under Section 3.2.2.

Pros

  • Faster computation due to lack of floating-point arithmetics (even multiplication and division are not necessary).

Cons

  • Limited to cases where the slope is between 0 and 1, and the starting point should be the leftmost point. (Some modification required to support such cases)

Usage

  • Move your mouse to change end point, left click to set start point of the line.
  • Select different algorithms and grid size to experiment.
